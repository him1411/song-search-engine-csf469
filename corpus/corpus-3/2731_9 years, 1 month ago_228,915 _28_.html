How would one create an iterative function (or iterator object) in python?Iterator objects in python conform to the iterator protocol, which basically means they provide two methods: __iter__() and next(). The __iter__ returns the iterator object and is implicitly called at the start of loops. The next() method returns the next value and is implicitly called at each loop increment. next() raises a StopIteration exception when there are no more value to return, which is implicitly captured by looping constructs to stop iterating.Here\'s a simple example of a counter:This will print:This is easier to write using a generator, as covered in a previous answer:The printed output will be the same.
